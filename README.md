# Statistical-A-B-Testing-Suite
This project implements a complete A/B testing analysis workflow to evaluate differences between control and treatment variants. It demonstrates statistical reasoning, experiment simulation, and decision-focused visualization.
Key Highlights
	•	Implemented experiment simulation and conversion analysis
	•	Applied hypothesis testing and confidence interval evaluation
	•	Built reproducible notebook workflow for experiment analysis

Technical Stack

	•	Pandas
	•	NumPy
	•	Matplotlib / Seaborn
	•	Jupyter Notebook

Workflow
